Software Altova

Somos partner de Altova, por lo que varios de nuestros productos se desarrollan con los distintos programas y software que Altova brinda y que respaldan componentes base de algunas de las soluciones que ofrecemos a nuestros clientes, como lo son XML, HTML y bases de datos como MySQL.

Altova Diffdog

Permite el seguimiento al desarrollo de documentos cuando se realizan proyectos complejos donde se deben tener versiones de documentos en forma controlada.

Altova UModel

Modelación de flujos de procesos, flujo de negocios, flujo de operaciones, diseño de bases de datos, diseño de operaciones, aplicación de flujo a proyectos de desarrollo de software y todos los flujos existentes para diagramar.

Altova StyleVision

Permite diseñar informes gráficos modernos y ágiles de cambiar por el usuario conectado a distintas fuentes de información aplicable a reportes XML, XBRL, Excel, Word, PDF, etc.

Altova XMLSpy

Herramienta que permite el desarrollo y gestión en HTML, XML, WSBL, XBRL, HTML5, CSS3, JavaScript y cualquier herramienta de desarrollo en Internet de forma fácil, amena y estructurada.

Altova DatabaseSpy

Administrador de bases de datos de alto desempeño con aplicación de Big Data y conexión con cualquier tipo de bases de datos: Oracle, MySQL, PostgreSQL, Access, entre otras.

Altova MapForce

Integración de las herramientas de desarrollo con fuentes de información de distinto tipo: bases de datos, Excel, Word, archivos planos, EDI y cualquier tipo de información almacenada.

Altova MobileTogether

El más ágil, fácil y rápido creador de herramientas para uso en dispositivos móviles indistintamente de su tipo, marca, tamaño y/o tecnología. Aplica video, sonido y el manejo de cualquier conexión con bases de datos y otras aplicaciones.
